## Description

Lemon Energy Test

Calculate Elegibility Prospect of new clients using pre-set rules.

[Nest](https://github.com/nestjs/nest) Framework

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# test coverage
$ npm run test:cov
```

## Swagger

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev
```

Then go to /api-docs
